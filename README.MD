# Boilerplate 
A starter boilerplate with webpack and ESLint config for JavaScript builds.

## Author
[Keith Rodriguez](http://www.keithrodriguez.me)

To Get Started Run:
npm install

### Start Development Server
```sh
$ npm run server
```

### Run Development Build
```sh
$ npm run dev-build
```

### Run Prodction Build
```sh
$ npm run prod-build
```
